How does Atom work with GitHub?

The process of publishing and sharing a webpage.

This process is super cool. The GitHub site holds tons of repositories, and is great for sharing webpages with large groups. This allows for colaborative projects. Follow these steps below to be able to share your live webpage:

  • 1. Make an account (or sign in) on GitHub.io, and create a repository.
  • 2. Download the GitHub desktop app and login.
  • 3. Go to the + button in the desktop app, and visit the clone tab.
  • 4. Find the respository you just made, and clone it. Pick a safe spot for this clone as it should never be moved!
  • 5. After you have cloned it, go to Atom, and create a new html file. Save it in the respository folder.
  • 6. Go back to the github desktop app, view your changes, and then add a summary.
  • 7. Click the publish button at the top, and it will begin syncing. Every time you commit the change and click sync, this will create a new version and your changes will no longer show.
  • 8. You can now go to Atom and view the page in Browser, and your changes will show!

Things NOT to do!

  • 1. Do not edit directly on the server on the GitHub page.
  • 2. Do not delete any folders from the repository.
  • 3. Only sync when you need to. View locally otherwise.

How do you open your files each time?

  • 1. Go to your github account online, and find the repository.
  • 2. Click on download, open in desktop. Launch the applications.
  • 3. In the github desktop app, find your repository and either open it directly, or go to Atom and locate it to open it.
